# Проект часть 2 – OpenMP-2

## (4) Ньютон.

Напишите программу, которая выполняет интерполяцию произвольной функции, заданной
таблично, по полиному Ньютона. Для распараллеливания использовать OpenMP, использовать весь
ресурс параллелизма (параллелить, где можно) и оптимизации компилятора. Проверить на различных
функциях (не менее 5 тестов). Построить график зависимости ускорения многопоточной программыпо
сравнению с 1 потоком и сравнить с линейным ускорением.
